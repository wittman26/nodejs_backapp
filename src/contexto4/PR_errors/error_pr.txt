Obtener el titular, documento de identificación del titular y el centro del expediente. 

Si productId = ‘AN’, ‘IN’, ‘PC’ o ‘PS’ → Buscar los datos en la BBDD, esquema ACELER, 

Para el titular, nombre y documento de identificación del titular y centro: tabla ACE_EVENTO campos TITULAR, NOMBRE_TITULAR, DOCUMENTO_TITULAR e  IDCENT cuando ID_ENTIDAD= X-ACELERA-ENTITY (header), ID_EVENTO= originId (path)

Else:

Si productId = AC → Buscar los datos en la BBDD, esquema ACELER, 

Para el titular, nombre y documento de identificación del titular: tabla ACE_ACUM_CLIENTE campos HOSTID, NOMBRE y DOCUMENTO cuando ID_ENTIDAD= X-ACELERA-ENTITY (header), ID_OPERACION= originId (path)

Para el centro: tabla ACE_ACUM_OPERACION campo IDCENT cuando ID_ENTIDAD= X-ACELERA-ENTITY, ID_OPERACION= originId (path)

Si productId != AC → Buscar los datos en la BBDD, esquema ACELER, 

Para el titular y centro: tabla ACE_OPERACION campos TITULAR e IDCENT cuando ID_ENTIDAD= X-ACELERA-ENTITY (header), ID_OPERACION= originId (path)

Para el nombre y documento de identificación del titular: tabla ACE_OPERACION_TITULARES campos NOMBRE y DOCUMENTO cuando ID_ENTIDAD= X-ACELERA-ENTITY (header), ID_OPERACION= originId (path)
